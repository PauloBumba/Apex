JuvsConnect : nome escolhido pois conecta os estudantes com os jogos universitários de um modo interativo

O software apresenta diversas que são uteis para o usuario e tanto pra os admistradosres permitindo que usuarios faça o login e tenha acesso ao site onde serão postatados os resultados , calendarios tudo em paginas divididas para cada área as transmissões do juvs 2023 foram um novidade , a impletação do site para gerenciar esses jogos terão as mesma ideia trazendo assim mais telepescatados no JUVS

Os jogos seriam actualizado em tempo pelo usuario e também a implementação de JavaScrits para mais interação o Presente Projeto como chatbox para que cada usuarios tem acesso a dar suas opinião, o presente projeto foi desenvolvido pelo Estudantes:
